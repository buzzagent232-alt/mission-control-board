Important Note

This GitHub Pages site cannot directly read local files from your Mac such as /Users/scottwalker/.openclaw/....

To display live local OpenClaw data, you will need either:

  • a local web server on your Mac, or
  • an export process that writes JSON into this repo and pushes updates.
